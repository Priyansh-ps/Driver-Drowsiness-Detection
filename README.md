# Driver Drowsiness Detection 

A real-time Driver Drowsiness Detection System built using Python, OpenCV and MediaPipe.

## Project Structure
```text
Driver-Drowsiness-Detection/
│
├── backend/
│   ├── eye_landmark.py
│   ├── face_detection.py
│   ├── requirements.txt
│
├── frontend/
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── .gitignore
├── requirements.txt
└── README.md
```
## Features

- Real-time face detection
- Eye landmark detection
- Eye Aspect Ratio (EAR)
- Drowsiness alert
- Modern dashboard UI (In Progress)

## Technologies

- Python
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript
